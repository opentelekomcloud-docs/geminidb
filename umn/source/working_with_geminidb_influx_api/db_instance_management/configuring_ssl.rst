Configuring SSL
===============

After a GeminiDB Influx instance is created, you can enable or disable SSL.

Usage Notes
-----------

-  After enabling or disabling SSL, restart the DB instance for the change to take effect.

.. _nosql_05_0151__en-us_topic_0000001165885055_section13582111184815:

Enabling SSL
------------

#. :ref:`Log in to the GeminiDB console <nosql_login>`.
#. On the **Instances** page, click the instance. The **Basic Information** page is displayed.
#. In the **DB Information** area, click |image1| to enable SSL.

.. _nosql_05_0151__en-us_topic_0000001165885055_section151208517481:

Disabling SSL
-------------

#. :ref:`Log in to the GeminiDB console <nosql_login>`.
#. On the **Instances** page, click the instance. The **Basic Information** page is displayed.
#. In the **DB Information** area, click |image2| to disable SSL.
